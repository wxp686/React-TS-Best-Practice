# REACT + TS + ANTD-PRO + git 约束

## Node Version

```bash
#建议使用nvm管理node版本
^10.13.0 || ^12.13.0 || >=14.0.0
```

## Environment Prepare

Install `node_modules`:

```bash
yarn
#or
npm install
```

## Start project

```bash
yarn start
#or
npm start
```

## Build project

```bash
yarn build
#or
npm run build
```

## git commit 规范

项目配置文件 commitlint.config.js

```bash
git commit -m 'type(scope): subject'
```

> 更多查看 [commitlint](https://commitlint.js.org/#/reference-rules)
